# 🗡️ Adventure Game

A 2D adventure game built entirely in **Java** — no game engine, just pure Java with custom rendering and game loop logic.

---

## 📖 About

This project is a 2D adventure game developed from scratch using Java. Without relying on any external game engine, the game handles its own rendering pipeline, game loop, collision, and resource management using Java's standard libraries (AWT/Swing).

---

## 🚀 Getting Started

### Prerequisites

- Java JDK **8 or higher**
- An IDE such as [IntelliJ IDEA](https://www.jetbrains.com/idea/) (recommended — the project includes `.idea` config files)

### Installation & Running

1. **Clone the repository**
   ```bash
   git clone https://github.com/atharvanalbalwar-a11y/Adventure-Game.git
   cd Adventure-Game
   ```

2. **Open in IntelliJ IDEA**
   - Open the project folder directly — IntelliJ will detect the `.iml` file and configure the project automatically.

3. **Run the game**
   - Locate the `main` class inside the `src/` directory.
   - Click the green **Run** button, or right-click the main class and select **Run**.
